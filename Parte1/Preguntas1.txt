¿Cómo podemos crear un repositorio remoto?

Primero tenemos que crear un repositorio local, despues de eso para realizar
un repositorio remoto hay que escribir el siguiente comando

git remote add origin https://github.com/(tu-username)/(nombre-repositorio).git

Y si queremos actualizar nuestro repositorio con nueva información deberiamos
usar un 'push', que sería el siguiente comando

git push -u origin main
